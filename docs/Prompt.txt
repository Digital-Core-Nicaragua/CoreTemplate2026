Qué es CoreTemplate
Plantilla reutilizable para sistemas ASP.NET Core 10

Multi-tenant configurable (IsMultiTenant: true/false en appsettings)

Autenticación completa con JWT + Refresh Tokens + 2FA TOTP (configurable)

Roles y permisos con formato Modulo.Recurso.Accion

Soporte SQL Server y PostgreSQL

Al terminar habrá un script rename.ps1 que reemplaza CoreTemplate por el nombre del nuevo sistema
